# Fretron

Fretron is a trucking and route-sharing app for transporters and businesses.

## Features

- Browse freight routes
- Create transporter routes
- Switch between transporter and business modes
- Review bookings and dashboard activity

## Tech Stack

- Frontend: React (Not entirely made by us implement code from uiverse and react bits which seemed nicer )
- Backend: Node.js and Express (cuz we only know a little bit of this)
- Auth and data: Supabase (cuz free and peak)

## Run Locally -(We will host it later on vercel so ez)

From the project root: 

```bash
npm run dev
```

## Notes

- The app uses a route-themed blue and purple animated background.
- Admin login is handled with a static demo credential in the backend.
